# 👨🏻‍💻 Customer Behavior Data Analysis Project

## 📌 Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI.
The goal is to understand customer purchasing patterns and generate business insights for decision-making.

---

## 📊 Objectives

* Analyze customer demographics and purchase behavior
* Identify top-performing product categories
* Understand spending patterns across segments
* Build an interactive Power BI dashboard
* Provide data-driven business recommendations

---

## 🛠️ Tools & Technologies

* Python (Pandas)
* SQL (Data querying & analysis)
* Power BI (Dashboard creation)
* Excel / CSV (Dataset handling)

---

## 📌 Key Insights

* Customer spending varies significantly across product categories
* Certain customer segments contribute more to total revenue
* Seasonal trends impact purchasing behavior
* Discount usage influences purchase frequency
